## Быстрые команды

### Запуск локально
```bash
python main.py
```

### Docker
```bash
docker compose up --build -d
docker compose down
```

## Суб-скиллы

Используй для конкретных задач:
- `rocket-chat-dev` - Разработка, отладка, добавление фич
- `rocket-chat-deploy` - Деплой, Docker, конфигурация
